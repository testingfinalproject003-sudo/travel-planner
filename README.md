## 🌍 Travel Planner — Plan Trips Together
A beautiful, feature-rich Flutter application for planning trips with friends. Built with Firebase backend, real-time chat with voting, AI-powered travel advice, interactive maps, and social features.
## ✨ Features
- 🗺️ Trip Planning
- Create Trips with destination, dates, notes, and budget
- Day-by-Day Itinerary with activities and scheduling
- Activity Suggestions with voting system (upvote/downvote)
- Trip Status Tracking — planning → confirmed → completed
- Reuse Past Trips — clone previous trips with one tap
# 💬 Real-Time Chat & Voting
- Private & Group Chats — auto-created on trip creation
- Trip Proposals — send trip plans to chat for group voting
- Location Sharing — share current location in chat
- Online Status — see who's online in trip chats
- AI-Powered Suggestions — get travel advice from OpenRouter AI
 # 🗺️ Maps & Explore
- Interactive Maps with Flutter Map (OpenStreetMap)
- Nearby Places via Foursquare API
- Distance Calculation between current location and destination
- Directions — open Google Maps for navigation
- Location Picker — search and select locations on map
# 👥 Social Features
- Add Friends via email/username
- Friend Requests with accept/decline
- Auto Chat Creation — private chat on friend accept, group chat on trip create
- Member Confirmations — track who's confirmed for the trip
# 🌤️ Weather & AI
- Weather Forecasts for trip destinations
- AI Travel Advice — powered by OpenRouter (GPT/Claude models)
- Smart Recommendations based on dates and destination
# 👤 Profile & History
- User Profile with avatar, name, email
- Trip History — view all completed trips
- Logout functionality
